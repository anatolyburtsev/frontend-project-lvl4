## Demo
[App on heroku](https://protected-headland-52299.herokuapp.com/)

## Requirements

* [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)

## Install

```sh
$ make install
```

## Run

```sh
$ make start
# open http://localhost:5000
```

## Deploy
```shell
$ heroku login
$ git push heroku main
```
